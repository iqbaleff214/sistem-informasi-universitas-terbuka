# MSIM4310 - Analisis dan Visualisasi Data

Setelah mempelajari Mata kuliah Analisis dan Visualisasi Data MSIM4310 (2 sks) mahasiswa mampu menerapkan Prinsip Dasar Penyajian Data, Transformasi Data, Penanganan Data Hilang, Analisis Regresi, Visualisasi Data Teks, dan Visualisasi Data Interaktif. Mata kuliah ini membahas mengenai Prinsip Dasar Penyajian Data, Transformasi Data, Penanganan Data Hilang, Analisis Regresi, Visualisasi Data Teks, dan Visualisasi Data Interaktif. Mahasiswa belajar melalui diskusi, tugas serta praktikum mandiri. Ketercapaian hasil belajar diukur melalui diskusi, tugas tutorial, dan ujian akhir semester.
