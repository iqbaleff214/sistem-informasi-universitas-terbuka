# STSI4105 - Basis Data

Mata kuliah ini secara umum membahas tentang konsep, pengembangan dan penggunaan basis data. Sedangkan materi yang dibahas dalam mata kuliah ini adalah Model Data, Entity Relationship Diagram (ERD), Normalisasi, Structured Query Language (SQL), Manajemen Transaksi, Konsep Basis Data Terdistribusi, dan aplikasi sistem basis data.

Setelah mempelajari basis data ini, mahasiswa mampu mengorganisir data yang tersimpan secara elektronik pada sebuah sistem komputer sehingga menghasilkan informasi yang berguna.
